Requirements:
Android Studio.
Minimum 3 android devices.

1. Open Android Studio -> Open an Existing Android Studio Project -> GoWifi(Locate it in the downloads folder after Unzipping it).Make sure you have all the necessary SDK  platforms.

2. Run the application on three devices.

3. One device will be the server and other two devices will be client.

4. Connect the two devices to the server by using bluetooth.
